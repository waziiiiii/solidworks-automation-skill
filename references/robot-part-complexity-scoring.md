# 机器人零部件 Codex 自动化建模复杂度评分

用途：在 AutoLife SolidWorks 建模、修改或装配任务开始前，先判断零部件是否适合交由 Codex + SolidWorks 自动化先行建模。评分只判断“自动化建模可行性”，不等同于最终工程交付风险。

## 必须执行的评分门禁

对任何零件/装配体建模请求，先完成以下步骤：

1. 读取用户提供的文字、图片、草图、旧模型、尺寸、装配约束和用途。
2. 按本文件的 8 个维度逐项评分并计算总分。
3. 给出等级、理由、Codex 可承担内容、需要人工确认内容。
4. 明确询问用户是否按该等级继续建模。
5. 只有用户确认继续后，才运行 SolidWorks 自检、生成脚本、打开/修改模型、保存或导出文件。

若用户只询问评分、评审、排障、解释 skill 或更新文档，不需要等待建模确认；若用户要求立即建模，也仍然先评分并等待确认。

## 主评分表

| 序号 | 评分维度 | 权重 | 评分提示 |
|---|---:|---:|---|
| 1 | 输入/比例参考完整度 | 5 | 尺寸完整 0；可按比例建模 2；关键接口缺失 4；完全无参考 5 |
| 2 | 几何规则度/曲面复杂度 | 15 | 规则板块/轴孔低分；复杂外壳、自由曲面高分；触发否决规则时按规则升级 |
| 3 | 特征数量与依赖 | 15 | 基础特征少低分；阵列、筋、壳、多基准、父子依赖多则高分 |
| 4 | 设计意图/参数化 | 15 | 固定尺寸低分；需要公式、配置、接口联动、轻量化判断则高分 |
| 5 | 装配关系/接口 | 15 | 单零件低分；电机、轴承、减速器、传感器、多配合接口高分 |
| 6 | SolidWorks 自动化稳定性 | 15 | API 可稳定拉伸/切除/阵列低分；依赖交互选面、曲面修补、未封装 API 高分 |
| 7 | 受力/运动风险 | 10 | 静态安装件低分；高载荷、冲击、扭矩、运动避让高分 |
| 8 | 精度/公差要求 | 10 | 普通间隙低分；同轴度、轴承位、定位销、压配/滑配、公差链高分 |
|  | 合计 | 100 |  |

## 等级判定

| 等级 | 总分 | 建模策略 | 适用结论 |
|---|---:|---|---|
| L1 | 0-30 | Codex 可完全完成 | 适合直接自动建模、保存、导出，并做预览自审查。 |
| L2 | 31-55 | Codex 建模 + 人工确认 | 可先自动建模，但需人工确认关键接口、尺寸或装配关系。 |
| L3 | 56-75 | Codex 只做辅助 | Codex 做草模、子零件、重复特征或工程图，主体设计由人工主导。 |
| L4 | 76-100 | 不建议 Codex 承担主体设计 | 仅做需求整理、局部脚本、文档、导出或辅助检查。 |

## 关键 15 分维度细分

### 几何规则度/曲面复杂度

| 子项 | 分值 | 打分提示 |
|---|---:|---|
| 主体几何 | 0-6 | 板、块、轴、孔、槽为 0-2；壳体过渡为 3-4；复杂自由曲面为 5-6 |
| 曲面质量要求 | 0-5 | 无曲面 0；普通圆角/过渡 1-2；外观面/人机曲面 3-5 |
| 可脚本表达程度 | 0-4 | 能由拉伸/旋转表达 0-1；需扫描/放样 2-3；需人工调面 4 |

### 特征数量与依赖

| 子项 | 分值 | 打分提示 |
|---|---:|---|
| 特征数量 | 0-5 | 1-5 个 0-1；6-12 个 2-3；13-25 个 4；25+ 个 5 |
| 特征类型 | 0-3 | 拉伸/切除/孔 0-1；阵列/镜像/筋/壳 2；复杂曲面特征 3 |
| 基准/父子依赖 | 0-4 | 单基准 0；多基准 2；复杂参考关系或易断父子关系 4 |
| 重建风险 | 0-3 | 稳定 0；局部可能失败 1-2；顺序敏感/重建风险高 3 |

### 设计意图/参数化

| 子项 | 分值 | 打分提示 |
|---|---:|---|
| 尺寸可调性 | 0-3 | 固定尺寸 0；部分可调 1-2；大量尺寸参数驱动 3 |
| 参数关联逻辑 | 0-4 | 无公式 0；简单关联 1-2；孔距、壁厚、间隙、比例联动 3-4 |
| 配置/系列化 | 0-3 | 单版本 0；少量变体 1-2；多配置系列件 3 |
| 工程判断含量 | 0-5 | 按图建模 0；需判断间隙/强度方向 2-3；需轻量化/刚度/寿命判断 4-5 |

### 装配关系/接口

| 子项 | 分值 | 打分提示 |
|---|---:|---|
| 接口数量 | 0-4 | 无接口 0；1-2 个 1；3-5 个 2-3；6+ 个 4 |
| 接口类型 | 0-4 | 普通螺丝孔 0-1；电机/传感器/型材 2；轴承/减速器/定位销 3-4 |
| 配合精度 | 0-3 | 普通安装 0；对齐/同轴 1-2；压配/滑配/公差链 3 |
| 运动/干涉关系 | 0-4 | 静态 0；简单转动/滑动 2；多自由度/线束避让/干涉敏感 3-4 |

### SolidWorks 自动化稳定性

| 子项 | 分值 | 打分提示 |
|---|---:|---|
| API 稳定性 | 0-6 | 常规拉伸/切除/孔 0-2；需少量未封装 API 3-4；依赖复杂 API 5-6 |
| 人工交互依赖 | 0-5 | 无需交互 0；少量选面确认 1-2；需手工修补/调面 3-5 |
| 审查可确认性 | 0-4 | 预览图易检查 0-1；需装配/干涉检查 2-3；难以视觉确认 4 |

## 5 分与 10 分维度评分

| 维度 | 权重 | 打分提示 |
|---|---:|---|
| 输入/比例参考完整度 | 5 | 尺寸完整 0；主要尺寸完整 1；有比例参考可先建模 2；只有结构描述 3；关键接口缺失 4；完全无参考 5 |
| 受力/运动风险 | 10 | 静态低载 0-2；一般支撑/普通运动 3-5；高载荷/冲击/扭矩 6-8；运动安全敏感 9-10 |
| 精度/公差要求 | 10 | 普通间隙 0-2；普通定位 3-5；同轴/轴承位/定位销 6-8；压配/滑配/公差链 9-10 |

## 一票否决

- 复杂自由曲面、A 面外观、人机曲面或高度拟真外壳若无法通过稳定 API 生成，直接判为 L4。
- 复杂自由曲面、A 面外观、人机曲面或高度拟真外壳若需要大量人工调面，最低 L3。

## 工程风险标签

以下标签不计入主分，但需要在评分结论中提醒用户：

- 需结构强度验证
- 需运动/干涉验证
- 需热/散热验证
- 需工艺评审
- 需测试部门验证
- 需安全/认证审查
